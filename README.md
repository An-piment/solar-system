## PARA INICIAR O PROJETO

Para instalar as dependências do projeto utilize o comando:

```sh
npm install
```
Para rodar o projeto utilize o comando:

```sh
npm start
```

## DEPLOY DO PROJETO

Você pode visualizar a página em:

```sh
https://solar-system-iota-rose.vercel.app/
```
